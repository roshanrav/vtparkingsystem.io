Our chosen system is a new and improved design for the Virginia Tech parking system. The system can show available parking spots on campus in real time to users. There is currently a parking system in place on campus at Virginia Tech that has already been completed, but we wish to create a new system to move it from a physical to virtual process. Our goal is to take current parking meters and create a mobile application for drivers to use instead, so the design choices we make will be original and new for our specific goal. This will allow teachers, students, and visitors to locate available parking spots, reserve space in advance, and get real-time parking updates. We hope to create a system with a modern user interface that allows for both traditional quarter and cash payment as well as credit card functionality.
